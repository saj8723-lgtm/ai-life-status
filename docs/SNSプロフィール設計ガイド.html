Canvaで作る手順（ヘッダー＆アイコン）

ヘッダー（10分）：

1. Canva → 「Twitterヘッダー」テンプレを選択（1500×500）

2. 背景を #faf5ff → #e8ddf5 のグラデーションに

3. 上端にレインボーのライン（3px）を配置

4. 中央に「⚔️ AI人生ステータス診断」とDotGothic16風のテキスト

5. サブテキスト「名前を入れるだけ。人生がRPGになる。」

6. 背景に薄くステータスEmoji（💪🧠💰😎🔥🍀）を散りばめる

アイコン（5分）：

1. Canva → 「ロゴ」テンプレを選択（500×500）

2. 円形背景 #f5f0ff

3. 中央に ⚔️ Emoji（大きく）

4. PNGで書き出し
